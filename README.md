# Tomato:bit

Extension which is suitable for Robot:bit and mBridge from KittenBot, including some common components & sensors:
* LCD Screen
* DHT11 Temperature & Humdity Sensor
* Soil Moisture Sensor
* Water Sensor
* PIR
* Ultrasonic sensor
* RGB LEDs @ Robot:bit
* Buzzer @ Robot:bit
